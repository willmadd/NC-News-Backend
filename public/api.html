End Points

GET /api/topics

Brings back a list of all available topics on Northcoders News

GET /api/topics/:topic_slug/articles

Return all the articles in a topic e.g: `/api/topics/coding/articles`

POST /api/topics/:topic_slug/articles

Posts a new article to the specified topic! This should be a JSON object in the following format
{
title: "Title Goes here",
body: "Article Content goes here"
}

GET /api/articles

Brings back all the articles

GET /api/articles/:article_id

Returns an article when given an id e.g./api/articles/ehjk123kjwdsa

GET /api/articles/:article_id/comments

Returns all the comments associated with an article

POST /api/articles/:article_id/comments

Adds a new comment to an article, this should be in the format
{
"body": "This is my new comment",
"created_by": user_id
}

PUT /api/articles/:article_id

This increases or decreases the vote count of an article by 1. It should be used with the parameters vote=up or vote=down e.g. /api/articles/8asdf98hjsdfk?vote=up

PUT /api/comments/:comment_id

This increases or decreases the vote count of an article by 1. It should be used with the parameters vote=up or vote=down e.g. /api/comments/9dasjk8dasd9d?vote=down

DELETE /api/comments/:comment_id

Deletes a comment by it's ID e.g. /api/comments/sadf9asdf90f

GET /api/users/:username

Returns all the info about a specified user!
